Figma Assignment

This project is a responsive frontend website built using Next.js and Tailwind CSS based on the provided Figma design.

Live Demo:
https://figma-assignment-ten-psi.vercel.app/

Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel

Features

- Fully responsive design
- Desktop layout
- Tablet layout
- Mobile layout
- Mobile sidebar menu
- Hero section
- Features section
- Responsive navigation bar

Setup Instructions

1. Clone repository

git clone YOUR_GITHUB_REPO_LINK

2. Install dependencies

npm install

3. Run development server

npm run dev

4. Build production

npm run build

Project Structure

- app/page.tsx
- public/
- README.md

Responsive Breakpoints

- Desktop
- Tablet
- Mobile

Deployment

The project is deployed on Vercel.
